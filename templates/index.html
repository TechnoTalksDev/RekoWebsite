What is Reko?

Reko is modern Minecraft Server Status bot. Well what is a Minecraft Server Status bot? Glad you asked! Reko allows you to check stats of your favorite Minecraft Servers! For example you can quickly see how many people are on your personal SMP! Or maybe you want to make sure your server isn't getting to laggy and want to check the ping to your server. Maybe even check that your server is on the right version! The list continues! I'm sure you'll find something of value!
For the entire list run /help with the bot!

Features

  • Customize for your server!

    Set a simple command (/server) to give the status of your custom set server! More features like updating charts for your set server are coming soon!

  • Fast status checks!

    Instantly check the status of a server with simple commands!
    Quick, simple, and easy!

  • Ready out of the box!

    Add Reko to your server and immediately be able to check status's of your favorite server's! Ready out of the box with no setup!
